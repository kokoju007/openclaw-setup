OpenClaw가 뭔가요?

일반 앱과 완전히 다른 방식으로 동작합니다. 이 개념만 잡으면 나머지는 쉬워요.

🖱 일반 프로그램

아이콘 더블클릭 → 직접 조작 → 창 닫으면 종료. 내가 움직여야만 작동합니다.

🤖 OpenClaw

아이콘 없음. 창 없음. 설치하면 보이지 않는 곳에서 24시간 대기합니다.
텔레그램으로 말 걸면 즉시 응답. 서버가 켜져 있는 한 항상 깨어 있어요.

내 텔레그램 ──────→ VPS (OpenClaw 상주) ──────→ Claude AI 처리
"해줘" 한 마디 명령 전달 즉시 응답
💡
VPS = 외부 사무실. 내 노트북을 꺼도 사무실 직원(OpenClaw)은 계속 일해요. 텔레그램은 그 직원에게 연락하는 메신저입니다.
02
VPS 서버 구매 (Contabo)

OpenClaw가 24시간 돌아갈 서버가 필요합니다. 가성비 최고인 Contabo 기준으로 설명합니다.

1
contabo.com 접속
https://contabo.com 접속 → 상단 메뉴 Cloud VPS 클릭
2
플랜 선택
CLOUD VPS 1 (월 $6.99) 선택 — OpenClaw 혼자 쓰기에 충분합니다
3
OS 선택
Storage Type → SSD | Operating System → Ubuntu 22.04 선택
4
리전 선택
한국과 가까운 Singapore 또는 Japan 선택 (응답속도 빠름)
5
결제 & 가입
이메일 입력 → PayPal 또는 신용카드로 결제 → 가입 완료
⚠️ Contabo는 주문 먼저 → 결제 후 가입 메일 발송 방식입니다
6
서버 정보 이메일 수신
결제 후 수분~수시간 내 IP 주소 + root 비밀번호가 이메일로 옵니다
예시: IP 203.0.113.42 / PW xK9#mP2qR

💰 월 $6.99 (약 10,000원) — 이 서버에 OpenClaw를 올려두면 24시간 AI 에이전트가 작동합니다.

05
VPS에 접속하기 (SSH)

서버를 원격 조종하는 방법입니다. 딱 한 줄이에요.

🪟 Windows

Windows 키 + R → cmd 입력 → 엔터 → 검은 창(터미널) 열림

ssh root@서버IP주소
# 예시: ssh root@203.0.113.42
# (Contabo 이메일에서 받은 IP 입력)
🍎 Mac

⌘+Space → Terminal 검색 → 실행 → 위와 동일하게 입력

⚠️ 비밀번호 입력 시 화면에 아무것도 안 보여요

정상입니다! 보안상 숨겨지는 거예요. Contabo 이메일의 비밀번호 그대로 치고 엔터 누르면 접속됩니다.

💡
접속 성공하면 root@vps:~# 형태의 프롬프트가 나타납니다. 이제 서버 안에 들어온 거예요.

원클릭 자동 설치

SSH 접속 후 아래 한 줄만 복사해서 붙여넣으면 끝입니다.

curl -fsSL https://raw.githubusercontent.com/kokoju007/openclaw-setup/main/install.sh | bash
💡
터미널 붙여넣기 → 마우스 우클릭 또는 Ctrl+Shift+V (Ctrl+V는 안 됩니다)

실행하면 자동으로 처리됩니다:

1
서버 환경 확인
Ubuntu 버전 자동 감지 — 20.04 / 22.04 / 24.04 지원
2
Node.js 자동 설치
이미 있으면 스킵, 없으면 공식 저장소에서 자동 설치
3
OpenClaw 설치
npm 공식 패키지 설치 (npmjs.com 등록 패키지)
4
텔레그램 봇 연결
봇 토큰 + 사용자 ID 입력 → 설정 파일 자동 생성
5
보안 설정
설정 파일 권한 잠금 (본인만 읽기 가능)
6
24시간 자동 실행 등록 ✅
서버 재부팅해도 자동 시작 — 서버가 켜져 있는 한 항상 대기
07

OpenClaw는 Claude AI를 두뇌로 사용합니다. API 키가 있어야 응답합니다.

1
console.anthropic.com 접속 → 회원가입
구글 계정으로 간편 가입 가능
2
API Keys → Create Key

3
텔레그램 봇에게 API 키 전달
설치 완료 후 봇에게 "API 키 설정"이라고 보내면 입력 방법 안내해줌

11
자주 묻는 질문
Q. 봇이 응답을 안 해요

① 봇 토큰이 맞는지 확인 → ② 내 텔레그램 ID가 맞는지 확인 → ③ sudo systemctl restart openclaw

Q. 서버 재부팅 후 작동 안 해요

sudo systemctl enable openclaw 실행 후 재시도

Q. SSH 접속이 안 돼요

Contabo 이메일의 IP/비밀번호 다시 확인. 비밀번호 입력 시 화면에 안 보이는 건 정상입니다.

Q. API 키 비용이 얼마나 나와요?

일반적인 개인 사용 기준 월 $1~5 수준. Anthropic 콘솔에서 한도 설정 가능합니다.
